# Apple Frameworks App #

## Description ##
ios app built with Swift UI, showing list of apple frameworks, detail, and link to official apple website.

### Techstack ###
- SwfitUI
- UI-Kit Controller Representable
- Reusable Components
- Foldering Structure
- @Binding and @State
- sheet
